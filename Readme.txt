Simple-Inventory-Management
Technologies: Node.js, ExpressJS, npm, Postman 
Project Overview: Create an inventory management system API where users can manage product stock levels. 
Features: Routes: GET /products: Get all products. 
GET /products/ : Get details of a specific product. 
POST /products: Add a new product. PATCH /products/ : Update product details or stock. 
DELETE /products/ : Delete a product. Data stored in an array. Handle status codes and responses for success/failure. 
Use middleware for logging and validation. 
Test using Postman.
